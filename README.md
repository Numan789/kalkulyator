# 🧮 Kalkulyator.net | Specialized Calculator Suite

A comprehensive collection of 20+ digital calculation tools designed for the Russian-speaking market. This project focuses on providing lightning-fast, accurate, and accessible mathematical, financial, and lifestyle utilities.

## 🚀 Overview

Kalkulyator.net is built as a highly optimized web application. It features a custom-built engine for mathematical precision and a modern user interface designed for high user engagement and retention.

<img width="1347" height="618" alt="bmi" src="https://github.com/user-attachments/assets/324dc685-9527-4d3e-8f20-2ba554a91efd" />

### Key Features
* **20+ Specialized Tools:** Covering algebra, geometry, finance, and daily health metrics.
* **Modern UI/UX:** Features a sleek glassmorphic aesthetic with isometric visual elements.
* **Performance First:** Zero-layout shift and sub-second load times via Next.js App Router.
* **SEO Optimized:** Full semantic HTML structure, JSON-LD schema markup, and optimized metadata for the Russian search landscape.
* **Responsive Design:** Fully adaptive layouts optimized for mobile users.

## 🛠 Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Deployment:** [Vercel](https://vercel.com/) / [Cloudflare](https://www.cloudflare.com/)
* **Icons:** Lucide React / Custom SVG Assets
